## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/guto8888/Space-tourism.git
```

Entre no diretório do projeto

```bash
  cd space-tourism
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
